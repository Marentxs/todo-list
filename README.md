A task management application built with vanilla JavaScript, bundled with Webpack, and designed to organize tasks into projects with persistent storage.

## Features
- Create, edit, and delete tasks
- Organize tasks into projects
- Task priority levels (changeable)
- Persistent data using localStorage
- Date handling and formatting with date-fns

## Built With

- JavaScript (ES6+)
- Webpack
- date-fns library
- HTML & CSS

## Getting Started
git clone https://github.com/Marentxs/todo-list.git

### Install dependencies
npm install

### Run development server
npm start

### Build for production
npm run build

## Deployment
This project is deployed using GitHub Pages and the `gh-pages` branch created from the `dist` directory.
